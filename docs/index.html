06 — Modeling Strategy

Three models, one
unified framework

A progressive modeling pipeline — from frequency baseline to Bayesian hybrid scoring to logistic regression severity prediction — each layer adding mechanistic depth.

Model 1

FAERS Baseline

Toxicity themes ranked purely by adverse event report frequency. Provides a strong statistical reference point but captures no biological reasoning — a frequency signal without mechanistic grounding.

FAERS Report Count → Theme Rank
Key Output High-frequency themes: Pneumonia, Hepatic Injury, Infections — reflecting reporting volume, not biological plausibility.
Model 2

Bayesian Path Scoring

Hybrid scoring combining FAERS frequency with knowledge graph path counts. Mechanistic paths are weighted by biological role, surfacing themes with strong pathway support that frequency-only ranking misses.

Drug → Protein → GO Process → Toxicity Theme
Key Finding Cell Death / Apoptosis and Renal Effects emerge as high-biology / low-reporting candidates — potential underreported signals.
Model 3 — Partner

Logistic Regression
Severity Prediction

Predicts whether a FAERS report corresponds to a serious adverse outcome using mechanistic features extracted from the knowledge graph. Validated with 5-fold cross-validation.

Mech. Features → Serious / Non-Serious
0.644 AUC ± 0.053
56.9% Accuracy ± 0.057
Key Finding Unique GO process count (mech_sum_unique_go) is the strongest positive predictor of serious outcomes — mechanistic diversity matters more than raw path count.
Technical Architecture

Nodes include Drug, Protein, GO Biological Process, and Toxicity Theme. Edges encode molecular binding, pathway involvement, and real-world safety associations. Model 2 uses Bayesian path scoring weighted by biological centrality. Model 3 extracts per-report mechanistic features (unique GO terms, unique proteins, weighted path scores, reaction counts) as inputs to L1-regularized logistic regression.

07 — Results

What the data shows

◈

Complementarity Analysis

KG path counts and FAERS report frequency capture orthogonal signals. Spearman ρ ≈ 0.18 — nearly uncorrelated. Themes like Cell Death, Cell Adhesion, and Oxidative Stress show strong mechanistic support despite low or zero FAERS frequency.

High-frequency categories such as "Other / Unmapped" lack mechanistic grounding, revealing reporting noise.
◉

Mechanistic Enrichment in Serious Reports

Serious adverse event reports show significantly higher mechanistic support than non-serious reports across all three mechanistic features.

Overall Interpretation

Ontology-based reasoning adds meaningful explanatory signal beyond simple reporting counts. Mechanistic features derived from the knowledge graph are statistically enriched in serious adverse event reports and contribute positively in severity prediction models — even when controlling for reaction frequency. The weak correlation between FAERS frequency and mechanistic path support demonstrates that these signals are non-redundant: frequency reflects reporting volume while ontology-grounded paths reflect biological pathway involvement tied to known molecular targets.

08 — Key Findings

What the framework reveals

  • I FAERS reporting frequency and mechanistic KG path support are nearly uncorrelated (ρ ≈ 0.18), validating the core premise of the neuro-symbolic approach — each signal captures a distinct dimension of toxicity risk.
  • II Cell Death / Apoptosis and Renal Effects are strong candidates for underreported adverse events — high biological plausibility, low reporting frequency — identified exclusively through the hybrid scoring approach.
  • III Serious adverse events are associated with broader and stronger biological pathway involvement (FDR-corrected p < 0.01), supporting the hypothesis that mechanistic complexity correlates with clinical severity.
  • IV Mechanistic grounding is not random noise — diversity-based features (unique GO processes, unique proteins) show the strongest predictive utility, outperforming raw path counts in severity classification.
09 — Impact & Limitations

Why this matters

Most AI systems in healthcare produce probabilities. This framework produces mechanistic explanations — bridging molecular biology and patient-level outcomes.

Ontology-grounded reasoning improves transparency in drug safety analysis, enabling clinicians and researchers to interrogate the basis of any prediction.
